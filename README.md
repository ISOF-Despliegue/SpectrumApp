# Spectrum App

## Descripción del Proyecto
Spectrum es una plataforma integral diseñada para la comunidad de videojuegos, que funciona como un espacio social para la exploración, calificación y reseña de títulos. El sistema permite a los usuarios interactuar con un catálogo dinámico, compartir sus opiniones y participar en módulos especializados como sistemas de votación y sorteos en tiempo real.

## Stack Tecnológico

### Frontend
* **React + TypeScript**: Para la creación de una interfaz de usuario moderna, segura y eficiente.
* **Electron**: Framework para el despliegue de la plataforma como una aplicación de escritorio nativa multiplataforma.

### Backend
* **API Principal (C#)**: Núcleo del sistema encargado de la lógica de negocio, autenticación y gestión de datos relacionales.
* **Microservicios (Java)**: Servicios especializados para el manejo de procesos concurrentes y comunicación mediante gRPC.

### Persistencia de Datos
* **PostgreSQL**: Motor relacional para el almacenamiento consistente de usuarios, juegos y reseñas.
* **MongoDB**: Base de datos documental para el manejo de datos de alta concurrencia y eventos dinámicos.

### Infraestructura y Calidad
* **Docker**: Orquestación de servicios mediante contenedores para garantizar la paridad entre los entornos de desarrollo y producción.
* **Husky & Commitlint**: Herramientas de automatización para asegurar el cumplimiento de estándares en el control de versiones.
